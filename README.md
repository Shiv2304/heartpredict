# HeartPredict

A comprehensive heart disease prediction system built with Python and an interactive web interface.

## 📋 Overview

HeartPredict is a machine learning-based application designed to predict heart disease risk using medical metrics and patient data. The project combines powerful Python backend processing with an intuitive HTML/CSS frontend for accessibility and ease of use.

## 🛠️ Technology Stack

- **Backend**: Python (39%)
  - Machine Learning & Data Processing
  - Model Training & Inference
  
- **Frontend**: HTML (58.2%) & CSS (1.3%)
  - Interactive Web Interface
  - User-Friendly Design
  
- **Other**: 1.5%

## ✨ Features

- 🏥 Heart disease risk prediction based on medical parameters
- 📊 Interactive web-based interface
- 🤖 Machine learning model for accurate predictions
- 📈 Data visualization and analysis
- 👥 User-friendly design for healthcare professionals and patients
